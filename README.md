A simple but powerful project that serves an `index.html` page using a custom Python socket server. It’s lightweight, educational, and a great way to understand how web servers work under the hood.

## ✨ Why this project is cool

- ⚡ **Minimal & fast** — no heavy frameworks, just pure Python socket power.
- 🧠 **Great for learning** — understand request handling and response flow.
- 🌐 **Web + backend combo** — HTML frontend served directly by Python.
- 🛠️ **Hackable foundation** — easy to expand into something bigger.

## 🧰 Tech Stack

- **Python** (Socket programming)
- **HTML** (Frontend page)

## 📁 Project Structure

.

## ▶️ Run Locally

1. Clone the repository:

```bash
git clone https://github.com/MaxKarltun/Python-Assign.git
cd Python-Assign
```

2. Start the Python server:

```bash
python server.py
```

3. Open your browser and visit:

```text
http://localhost:8080
```
